Desktop Interaction

Use this panel to quickly see inputs, percent rules, shortcuts, and advanced actions.

Quick Inputs

  • Brackets: ( and ) for grouped calculations
  • Power: use x^n button or ^ key
  • Sign toggle: use ± to convert + / -
  • Constants: p = pi, e = Euler value

% Calculation Inputs

  • 100-20% = 80
  • 200+10% = 220
  • 50×10% = 5
  • 400÷25% = 1600

Function Examples

  • sin(30) in DEG mode = 0.5
  • ln(e) = 1
  • log(1000) = 3
  • sqrt(81) = 9

Keyboard Shortcuts

  • Numbers: 0-9
  • Operators: + - * / %
  • Equals: Enter or =
  • Delete last: Backspace
  • Clear all: Delete or Esc
  • Functions: s c t l n r p e

Memory & Mode

  • MC: clear saved memory
  • MR: recall saved memory value
  • M+: add current result to memory
  • RAD/DEG: switch angle type for trig
